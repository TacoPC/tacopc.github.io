Say hi to Polyifed!

Meet Polyifed, the fun and friendly purple cat bot with a big fluffy tail!

Here is the discord server of Polyifed (Ployifed in server / you can add it to your server)

Here's a few examples of what Polyifed can do!

  • cat

    Get a random cat image using thecatapi.com

  • dog

    Fetch a random dog image using thedogapi.com

  • pet

    Pet Polyifed and see how happy it makes her!

There are more commands when you try Polyifed out!

Here's more information

Polyifed is a personal bot created by Tacopc, designed purely for fun.

It may have some bugs and is not intended for moderation.

While most commands are light-hearted and entertaining, some special commands like apply-as-dev are exclusive to certain servers and cannot be disabled from the commands menu.
